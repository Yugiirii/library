# T-Библиотека

**T-Библиотека** — это консольное приложение, написанное на языке Python, предназначенное для эффективного управления личной библиотекой книг. Приложение позволяет пользователю добавлять, просматривать, фильтровать, искать и управлять статусом книг, а также включает в себя рекомендательную систему, основанную на предпочтениях пользователя.

## Функциональные возможности

*   **Управление книгами:** Добавление, удаление и редактирование информации о книгах (название, автор, жанр, год издания, описание).
*   **Сортировка:** Возможность сортировки списка книг по различным критериям: название, автор, год издания, жанр.
*   **Избранное:** Добавление книг в избранное для быстрого доступа.
*   **Статус книги:** Изменение статуса книги на "прочитана" или "не прочитана".
*   **Фильтрация:** Фильтрация книг по различным критериям: название, автор, год издания, жанр и статус книги. 
*   **поиск:** Поиск по ключевым словам в названии, авторе или описании.
*   **Рекомендательная система:** Алгоритм, предлагающий книги, основываясь на предпочтениях пользователя, определённых через книги, добавленные в избранное. Популярность жанров и авторов, отмеченных пользователем, влияет на релевантность рекомендаций.
*   **Сохранение данных:** Все данные о библиотеке (включая статусы, избранное и предпочтения) автоматически сохраняются в формате JSON (`library.json`) и восстанавливаются при запуске приложения.

## Архитектура и технологии

*   **Язык программирования:** Python
*   **Архитектура:** Объектно-ориентированное программирование (ООП).
*   **Формат данных:** JSON для хранения информации о книгах.
*   **Алгоритмы:** Использование динамических структур данных для отслеживания популярности жанров и авторов, реализация рекомендательной системы на основе суммы весов.
